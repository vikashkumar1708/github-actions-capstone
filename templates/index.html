🔒 Security Tools & Scanning

Shift-Left Security

The pipeline integrates five automated security layers—Bandit (SAST), Gitleaks (Secrets Detection), pip-audit (Dependency Scanning), Hadolint (Dockerfile Validation), and Trivy (Container Image Security). These security checks are executed on every code push, ensuring that vulnerabilities are detected and addressed before deployment to AWS EC2.

🔬
SAST

Bandit

Python source code ki static analysis — SQL injection, hardcoded passwords, insecure deserialization jaise common vulnerabilities detect karta hai.

📏
Linting

Flake8

Python code style aur quality enforce karta hai — PEP8 compliance check, unused imports aur dead code pakadta hai.

🔑
Secrets Scan

Gitleaks

Poori git history scan karke accidentally committed API keys, tokens, passwords aur private keys dhundhta hai.

📦
Dependencies

pip-audit

Python packages ko PyPI Advisory Database ke against audit karta hai — known CVEs wale vulnerable packages block karta hai.

🐳
Dockerfile

Hadolint

Dockerfile ko best practices ke against validate karta hai — root user, latest tags, aur insecure instructions ke issues pakadta hai.

🛡️
Image CVE

Trivy

Built Docker image ke OS packages aur application libraries mein CRITICAL aur HIGH severity CVEs scan karta hai — vulnerable images deploy nahi hoti.
